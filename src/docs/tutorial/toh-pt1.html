The Tutorial Hero Editor

Demonstrates the CodeExampleComponent.

Enter the following command in the terminal window:

npm start

This command runs the TypeScript compiler in "watch mode", recompiling automatically when the code changes, launches the app in a browser, and refreshes the browser when the code changes.

Show the hero

Add two properties to the AppComponent.title property for the app name and a hero property for a hero named "Windstorm."

export class AppComponent { title = 'Tour of Heroes'; hero = 'Windstorm'; }

Don't forget the export!

class AppComponent { title = 'Tour of Heroes'; hero = 'Windstorm'; }
